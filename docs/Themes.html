jTable Themes

jTable has it's own themes. It also supports jQueryUI's theme system.

jQueryUI's theme system

jTable can use your jQueryUI theme. You can create and download your jQueryUI theme from http://jqueryui.com/themeroller/. So, limitless themes can be built for jTable easily. Here, there are some example jQueryUI themes. Click images to see running demos.
